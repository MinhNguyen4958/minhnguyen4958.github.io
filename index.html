Grooming Station

This assignment aims to fix a synchronization problem using mutexes and condition variables provided by the POSIX pthread library. A pet grooming facility has fixed amount of grooming stations, at each of which a single pet can be groomed at a time.
The owner imposes 4 rules on concurrency:

1. Cats and dogs should never be in grooming stations at the same time.

2. Only one pet can be occupying a grooming station at a time.

3. Niether cat nor dogs nor other pets should wait indefinitely to be groomed.

4. Concurrency solution must be efficient.

Developed in C with POSIX pthread library.
